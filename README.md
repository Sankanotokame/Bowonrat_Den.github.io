# Product Card Layout

## เป้าหมาย
ฝึกจัดข้อมูลให้เป็น Card สำหรับสินค้า/โปรเจกต์

## ไฟล์ในตัวอย่างนี้
`index.html`, `style.css`

## วิธีใช้งาน
1. เปิดโฟลเดอร์นี้ด้วย VS Code
2. เปิดไฟล์ `index.html` ด้วย Browser หรือ Live Server
3. ทดลองแก้ข้อความ สี หรือ Layout
4. ใช้ ChatGPT ช่วยอธิบายโค้ด แต่ต้องตรวจและแก้ไขด้วยตนเอง

## กิจกรรมสำหรับผู้เรียน
เปลี่ยนเป็นการ์ดนำเสนอผลงานของผู้เรียนเอง

## Prompt แนะนำ
```text
ช่วยอธิบายโค้ดในโปรเจกต์นี้แบบเข้าใจง่ายสำหรับนักเรียนมัธยมปลาย/ปวช. และเสนอจุดปรับปรุง 3 จุด โดยไม่เขียนแทนทั้งหมด
```
